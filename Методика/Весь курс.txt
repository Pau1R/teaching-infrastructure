Проведение всех занятий:
- контрольные работы каждый месяц с внесением в omni
- повторять примерно каждые две недели
- в день контрольной делать повторение теории перед работой

Повторение:
- ориентироваться на примерно каждые 2 недели
- маленькие темы объединять, если возможно. Срок в 2 недели можно подстраивать
- Если большая тема больше 4 недель, то сделать повторение посередине

Контрольные работы:
- блок с теоретическими вопросами (onlinetestpad.com)
- блок в практическими заданиями (github)

Типы занятий:
- освоение новых знаний
- выработка навыков (в т.ч. повторение)

Инструменты:
- xmind - сведение теории в единую систему
- github - выдача индивидуальных практических заданий, проверка
- onlinetestpad.com - тесты по теории
- sublime text - редактор кода (required extensions: terminus, smart path copy, live coding in python)
- визуализация исполнения:
  - sublime text live coding extension
  - pythontutor.com
