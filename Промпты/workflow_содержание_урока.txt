---
description: Генерация подробного содержания темы урока с четкой структурой для самостоятельного изучения
inputs:
  - название урока
  - краткая цель урока
  - список ключевых понятий
  - подробное объяснение всех элементов темы
  - примеры и разбор типичных ошибок
  - небольшие практические мини-задания
algorithm:
  - определить название урока и краткую цель
  - перечислить ключевые понятия и предоставить подробное объяснение
  - включить примеры и разбор типичных ошибок
  - добавить небольшие практические мини-задания
validation:
  - содержание должно быть самостоятельным и передавать тему урока студенту
  - структура файла должна соответствовать требованиям
output:
  - файл с подробным содержанием темы урока

Сгенерируй подробное содержание темы урока. Файл должен быть самостоятельным и полностью передавать тему урока студенту.

Требования:

1.  Структура файла:

    -   Заголовок с названием урока
    -   Краткая цель урока
    -   Список ключевых понятий
    -   Подробное объяснение всех элементов темы, с примерами и разбором
        типичных ошибок
    -   Небольшие практические мини-задания внутри текста
    -   Итоговый блок: что студент должен уметь после урока

2.  Пиши так, чтобы студент без контекста мог понять тему.

3.  Не добавляй домашние задания или практические задания из другой
    части системы. Только содержание темы.
